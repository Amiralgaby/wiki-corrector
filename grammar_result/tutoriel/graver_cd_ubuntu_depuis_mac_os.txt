39 Mot inconnu : overburn [[107:115]] => <note>Dans le cas d'Ubuntu 10.10, l'image iso pèse 710Mo. Or, l'**Utilitaire de disque** ne supporte pas l'overburn, soit la gravure de 710 Mo sur un disque de 700. Deux solutions : soit vous gravez sur un DVD, soit vous utilisez le terminal et la commande hdiutil. Dans ce second cas, ouvrez le terminal, et tapez <code>hdiutil burn </code> Sans appuyer sur "entrée", glissez-déposer le fichier iso de ubuntu dans la fenêtre du terminal. Vous devriez avoir quelque chose comme <code>hdiutil burn /le/chemin/vers/votre/iso/ubuntu-10.10-desktop-powerpc.iso</code>**Attention à l'espace après burn !** Vous pouvez appuyer sur entrée pour lancer la gravure. Et voilà !!! <|> overburn
