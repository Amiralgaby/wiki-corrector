8 Mot inconnu : ssh => Cependant certains réseaux ne laissent pas par défaut entrer les connexions ssh, le port par défaut (22) n'étant pas ouvert, comme ce peut notamment être le cas des particuliers derrière une BOX. Les connexions sortantes ne sont toutefois pas bloquées, ce qui va vous permettre de vous connecter quand même.

10 Mot inconnu : SSH => Une parade est donc d'utiliser la fonction REVERSE de SSH. Le principe consiste à initier une connexion depuis la machine derrière le routeur ((donc la machine dont le port entrant est bloqué)) sur une machine tierce ((celle-ci disposant d'un serveur SSH accessible)), et ainsi permettre une connexion retour depuis la machine tierce qui ne sera pas bloquée.
