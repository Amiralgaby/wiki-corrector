26 Mot inconnu : CDs [[169:172]] => Dans le premier cas comme dans le deuxième, la solution est simple : il faut graver une copie de sauvegarde dès réception de l'ordinateur. Bien noter où sont rangés ces CDs. Une solution évidente est de mettre l'un des deux dans le carton de l'ordinateur. Dans ce cas attention si vous rangez votre carton dans une cave ou dans un grenier : les conditions de température et d'humidité ne sont vraiment pas favorables à la conservation du CD. Prenez bien garde de ranger le deuxième CD dans un endroit à température ambiante sans exposition à la chaleur, le froid ou l'humidité (éviter le plein soleil et les radiateurs notamment), et faites en sorte de vous souvenir de l'endroit où vous l'avez rangé.  <|> cds
