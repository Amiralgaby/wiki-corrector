154 Mot inconnu : BootMgr => BootMgr est le nom donné à l'amorceur de Windows Vista. Celui-ci est différent des autres versions de Windows qui utilisent NTLDR. Par défaut, celui-ci est configuré pour démarrer automatiquement Windows, sans offrir de menu pour amorcer d'autres systèmes d'exploitation, à l'exception d'autres systèmes Windows, s'il y en a plus d'un, installé dans l'ordinateur. Par contre, BootMgr permet de //chaîner// un autre amorceur, qui lui a les capacités d'amorcer un système Linux ; c'est cette capacité qu'exploite ce tutoriel. | bootmgr
