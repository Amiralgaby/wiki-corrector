117 Guillemets typographiques fermants. => Chez certaines personnes le jeu plante dans le menu, ou lorsqu'on joue en ligne. En fait, c'est à cause des vidéos BIK (fichiers .bik). Pour régler ce problème, et pouvoir jouer en ligne, il suffit d'aller dans le répertoire de Trackmania et de modifier l'extension des fichiers .bik (par exemple .bik.bak) afin que TrackMania ne reconnaisse plus les fichiers vidéos et "zappe" donc plus les fichiers problématiques. <|> " => suggestions : ['\xa0»', '”']

121 Mot inconnu : Trackmania => Parfois il faut aussi désactiver les téléchargements dans les options avancées de Trackmania (dans le //Launcher//) pour qu'il ne télécharge pas automatiquement de nouvelles vidéos BIK. Pour jouer en ligne, il ne faut pas se connecter au lancement du jeu, sinon vous pouvez rencontrer un problème dans le menu à cause d'une vidéo BIK dans la //mania zone//. Il est préférable de se connecter au moment où vous cliquez sur Multijoueurs. <|> trackmania

Si le jeu devient lent dans le menu lorsque la vidéo BIK de la //mania zone// se charge, c'est normal, il vous suffit d'attendre quelques secondes.</note> <|> wine
