3 Mot inconnu : xbmc => ===== xbmc, prérequis =====
4 Mot inconnu : HID => [[:xbmc|xbmc]] est un logiciel afin de transformer votre ordinateur en [[:media_center|media center]]. Afin de piloter ce type de software, il est possible d'utiliser des claviers ou télécommandes. Un problème survenant assez souvent est l'absence de correspondance entre les touches permettant de naviguer à travers xbmc et les touches de votre télécommande. Cet article décrit donc les différentes étapes nécessaires pour utiliser un périphérique HID avec xbmc (ou tout autre logiciel media center).
