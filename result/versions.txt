92 Il manque un espace. => Contrairement à d'autres distributions %%GNU/Linux%%, lorsqu'une version d'Ubuntu est stabilisée, les versions des logiciels qu'elle inclut sont gelées. Ainsi, si une nouvelle version stable d'un logiciel ou d'une bibliothèque quelconque sort après la stabilisation d'Ubuntu, l'intégration de cette nouvelle version à Ubuntu se produit dans la mouture suivante du système, [[:installation#mise_a_niveau|à laquelle vous pouvez vous mettre à niveau sans réinstaller votre système en totalité]].

De plus, elle assure que le système, dans sa version actuelle, reste stable et fonctionnel.
